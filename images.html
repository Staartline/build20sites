Embedding Images using img src

We finally begin to learn about embedding images now. Pexels is a good resource for free images online.

Image Element -> This tag does not wrap anything. You give it the URL of an image that you would like to use, then close the tag. You can use external images from another site, or from the file structure of the website. Images use a SRC, or a source attribute.

The following photo uses img src, and an external website link

However, the image is quite large, no? Images can also have multiple attrubutes such as alt, for alt text. To reduce the size of this image, we can refer to W3Schools here to see that there is a width attribute, to reduce an image size. this is width. Width can take a 50% or a number in pixels.

It is recommended that a user will manually edit down the image size so that computers do not have to download the whole image.
