Hopefully Step by Step Instructions on how to make a One Page Site

This is a simple yet effective tutoiral that gets as much information out as possible

HOT TIP! Never use pure black. Use an Off black color.

  1. Set up the Boilerplate HTML
    • In the case you are using Sublime Editor, you can type in HTML, and it should automatically populate.
  2. Set up the DIVs on the page
    1. We want to set up the containers in the html to modify with CSS.
    2. In the case here, set up a DIV inside the body for the header.
    3. Inside the header div, set up a div for the logo
  3. Choose a specific image from Pexels according to the desired topic.
  4. Add a column
    1. create another div with an ID of threeColumns
    2. Create three divs within the div that contain different names for each column
  5. navigate to Lorem Ipsum, and copy in the text at this time
  6. Copy the columns to populate information in all the areas.
  7. Now, we are going to add in CSS.
    1. In the project folder, create a folder called css
    2. Inside the css folder, create a file called styles.css
    3. Inside the styles.css folder is where we are going to modify the site's look.
    4. Read more about CSS here -> here
    5. By the way, comments in CSS is written in a /**/ format, much like in Java
    6. Modifying items in CSS are default for h1 and h2 elements for example. clases must be designated by .classname
    7. Link in the CSS file or use css. There is Inline HTML, which uses CSS in the html elements. Internal using a style element in the head of the files, or external by using a link to link to the css file. This case uses an external css.

    Choose a HEX color here for a cute page https://www.w3schools.com/colors/colors_picker.asp

  8. Add in a DIV under the logo div for a banner div.
  9. Making the columns column
