Linking with A (anchor) tag

Linking can be used to link to other webpages, or to your own webpages. They use an a tag.

URLS can be absolute or relative

It is in the format of < a href="link">insert writing here < /a>, without the spaces.

Types of Linking

Absolute -> http://www.website.com for instance. External sites usually need http/s: in front of the link

Relative -> filename.html for websites in the same directory.

It is common to list some links, for instance, in navigation.

Attributes -> W3 Schools Link here

  • target -> Can be used to show where a link will open
