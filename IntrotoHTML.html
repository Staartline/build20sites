What is HTML?

Hypertext Markup Language

Used to make webpages -> HTML is a webpage's skeleton, and tells a browser how to display a page.

Tags and Elements

Such as H1, DIV and p are tags. They might be enclosed between tags such as: < tag> content < \tag>.

When something is enclosed in such a way, it is an element.

Boilerplate

Code found on every webpage.

Usually contains elements such as !Doctype, HTML, TITLE, BODY, and closing tags.

For example, visit example.com and view the page source. It should have the aformentioned tags.

!Doctype HTML - Pages need this to be rendered as HTML. Indicated to the browser.

HTML - Wraps all content on the page.

HEAD - This is where we put content that might not be shown to the site visitors. This information might include a link to a CSS sheet or Javascript files. Head might also include the following.

Meta charset - sets a character set for the website's encoding. UTF-8 Includes most characters.

Title - Sets the title on the webpage. Visitors can see this on their tabs.

Body- This is the content that is displayed on the website. This might include DIVs, H1, P, A, and so forth.

Order is important. Make sure to double check the nest, and close the tags off in a reverse order.
